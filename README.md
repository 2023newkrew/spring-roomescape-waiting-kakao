# Spring 방탈출 인증

## 기능 요구사항
- 인증기능 구현
- [x] 이름과 비밀번호로 토큰을 발급한다.
- [x] 토큰으로 유저 정보를 조회한다.
  - [x] 토큰으로 부터 유저 id를 추출한다.
- 예외 처리
- [x] username, password가 하나라도 잘못되면 로그인이 불가능 하다.
- [x] 인증이 필요한 API 호출시 토큰이 없으면 안된다.

## 추가 요구사항
- 예외처리
- [x] 예약하기 개선
  - [x] 비로그인 사용자는 예약이 불가능하다.
- [ ] 예약취소 개선
  - [ ] 자신의 예약이 아닌 경우 예약 취소가 불가능 하다.
